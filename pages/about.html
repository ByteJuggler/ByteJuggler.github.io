About this blog

This is a personal blog I've started for my work-related activities, which currently include mostly software development (primarily Delphi, Python and whatever seems to be required on the day to get the job done), some systems and database administration and some general IT consulting.

My background is mainly in IT, and I hold an M.Sc in IT and an undergraduate degree in Computer science and Applied Mathematics. In recent years however I've however developed an avid interest in computational finance and machine learning approaches to algorithmic trading and investment, and so am continuing to study and learn in this area. (In my spare time I work on my own algorithmic & quantitative trading models/strategies using Python and several libraries.)

This blog will serve to help keeping track of software projects I'm working on, useful how-to guides or other brain-dumps, and generally any technology or financial topics I find interesting.

Thanks for stopping by. :)
