First Blog Post!

By Walter Prins Date Wed 07 May 2014 22:57 Modified 2014-05-08 12:08 Tags python / pelican / blog

For a while now I've been toying with the idea to start a blog, to serve as a place to post brain dumps, rants, solutions and generally interesting content, but could never actually follow through and just get it done.

Today however (or rather, yesterday, 07/05/14), a colleague made a bet with me that I wouldn't be able to get my blog set up within 24 hours. To sweeten the deal we agreed that the loser would buy lunch. So, to prove him wrong and win the bet, I've finally set up this blog.

As for actually setting up, fortunately I've previously thought about this and had concluded that Pelican would be as good a choice as any. It turned out that Pelican and github made this job fairly easy, so here you go. My own Pelican powered blog hosted on the backend via Github pages. Thanks Pelican and thanks github! Heck, I might even do a blog post about how I set it up, now that I have a blog.

So, sorry Matt, but you lose. ;)
